## 简介
这是一个使用 Triton Inference Server 和前端 Vue/Vite 构建的深度学习推理与 RAG 系统项目。

## 包含的模块
- **frontend**: Vue 3 + Vite 前端界面
- **rag**: 检索增强生成 (RAG) 知识库构建与查询
- **train_model**: 模型微调与训练脚本
- **triton_model_repo**: Triton 推理服务器的模型仓库配置
- **相关脚本**: 用于导出 ONNX、批处理客户端测试、性能数据准备等
